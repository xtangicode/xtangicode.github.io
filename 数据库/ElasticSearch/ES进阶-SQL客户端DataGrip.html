    # IDEA

    官网地址：DataGrip：由 JetBrains 开发的数据库和 SQL 跨平台 IDE (opens new window)

    下载地址：下载 DataGrip：适用于数据库和 SQL 的跨平台 IDE (opens new window)

    # 下载对应版本Elasticsearch驱动到本地maven仓库

    <dependency>
    <groupId>org.elasticsearch.plugin</groupId>
    <artifactId>x-pack-sql-jdbc</artifactId>
    <version>7.8.0</version>
</dependency>

    # 选择驱动

    IDEA ->File->New->Driver->Elasticsearch Driver

    Name：Elasticsearch Driver

    General Class：org.elasticsearch.xpack.sql.jdbc.EsDriver

    Driver Files 添加刚下载的x-pack-sql-jdbc驱动，x-pack-sql-jdbc-7.8.0.jar

    # 新建数据源

    IDEA->File->New->Data Source->Elasticsearch Driver

    # 配置连接参数

    用户名和账号采用 ES 自带的 elastic 即可

    URL 地址 ：jdbc:es://http://localhost:9200

    如果使用HTTPS请求需配置SSL证书

    # 创建查询操作窗口 Query Console

    ElasticSearch Data Source ->New->Query Console

    show tables
报错：current license is non-compliant for [jdbc]

    # 更改 license 类型

    默认情况下，JDBC 客户端必须为白金级别才可以使用

    # 查看license 类型
GET _license

    这里可以将当前的 ES 软件的 License 暂时设置为试用版。测试完成后，改回 basic 版即可

    # 更改 License 类型 - trial
POST _license/start_trial?acknowledge=true
 
# 更改 License 类型 - basic
POST _license/start_basic?acknowledge=true

    再次执行show tables
